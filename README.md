# Sandales Artisanales – Site Vitrine

Site vitrine moderne pour présenter une collection de sandales faites main.  
Frontend en **Next.js 16**, backend en **Strapi v4**, design avec **Tailwind CSS**, animations avec **Framer Motion**.

## Fonctionnalités

- Catalogue dynamique des sandales (images, nom, prix)
- Présentation de l’artisan et de son savoir-faire
- Témoignages clients
- Formulaire de contact
- Contenu 100% géré depuis Strapi

## Technologies

- Next.js 16
- Tailwind CSS
- Framer Motion
- Strapi (CMS headless)
- Axios
- TypeScript

## Installation

### Frontend

```bash
npm install
npm run dev
```
